# Kiddo Prep – Quiz App
- Built with Next.js App Router, TypeScript, TailwindCSS, and shadcn/ui.
- Features timed quizzes, topic selection, results review, and heatmap analytics.

## Features
- Interactive question cards with instant feedback.
- Timers and time selection per quiz.
- Topic dashboard and progress heatmap.
- Detailed results and review mode.
- Theming via components/theme-provider.tsx.

## Tech Stack
- Next.js 14, TypeScript, TailwindCSS
- shadcn/ui components
- State + utils under lib/ and components/


- Node.js 18+
- pnpm (recommended)
``
pnpm install
``

``
pnpm dev
  Built with ❤️ by <a href="https://github.com/Anmol-rocket">Anmol</a>
``
App runs at http://localhost:3000.

### Build
``
pnpm build
pnpm start
``
